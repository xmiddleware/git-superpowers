## ğŸ¬ THE 6 LIVE DEMOS (In Order)

### Demo 1: Project Initialization (5 min)
```
- Create directory
- git init
- Create Main.java
- git add . && git commit
- Show git log
POINT: "This snapshot is immutable"
```

### Demo 2: Reflog Recovery (7 min) â­ MOST IMPORTANT
```
- Make a commit
- Reset hard (intentionally "break")
- Show git reflog
- Recover with reset --hard
- Prove it's back
POINT: "Nothing is ever truly lost"
```

### Demo 3: Detached HEAD (6 min)
```
- Checkout old commit
- Modify code
- Create branch to save (or discard)
- Show safety
POINT: "Exploration playground"
```

### Demo 4: Reset/Revert/Checkout (8 min)
```
- Show same mistake
- Fix with reset
- Fix with revert
- Fix with checkout
POINT: "Three tools, different contexts"
```

### Demo 5: Branches Are Cheap (5 min)
```
- Create multiple branches
- Break code intentionally
- Delete branches
- Show commits in reflog
POINT: "Free to experiment"
```

### Demo 6: Stash & Cherry-pick (4 min)
```
- Stash work mid-stream
- Pop it back
- Cherry-pick a commit
POINT: "Flexibility and precision"
```
